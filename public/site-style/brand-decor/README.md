# Brand Decor & Site Style Assets

Welcome to your Site Style directory! This folder is dedicated to hosting brand-specific decor, tile patterns, vector icons, and transparent brand assets that are used to style the website.

You can upload, add, or replace any files here to customize the look and feel of the site.

## Included Files:
- `/public/site-style/brand-decor/taco-decor.svg` - Hand-crafted neo-brutalist style taco vector.
- `/public/site-style/brand-decor/burger-decor.svg` - Hand-crafted neo-brutalist style burger vector.
- `/public/site-style/brand-decor/chili-decor.svg` - Hand-crafted chili pepper vector.
- `/public/site-style/brand-decor/starburst-decor.svg` - Neo-brutalist style starburst/sparkle asset.

Feel free to add your own brand tiles and decorations right here!
